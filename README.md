# Primer


## Installation

```console
git clone https://github.com/yliniemi/primer
```


## Compilation

### Prerequisites for Debian 12 and above

```console
sudo apt install cmake pkg-config
```


### Actual compilation

```console
git clone https://github.com/yliniemi/primer
cd primer/build/
cmake ..
make
./primer
```


## Interfacing

### Command line arguments

Example
```console
./primer
```
